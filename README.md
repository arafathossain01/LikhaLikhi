# ছন্দ (Chondo)

A simple Bangla poetry/chondo viewer built with HTML, Tailwind CSS, and JavaScript.

## 🛠️ Run Locally

1. Install dependencies (Tailwind CLI):

```bash
npm install
```

2. Build the CSS (if you modify Tailwind inputs):

```bash
npx tailwindcss -i ./src/input.css -o ./dist/output.css --watch
```

3. Open `index.html` in your browser.

## ✨ Features

- Simple responsive layout
- Next/previous navigation buttons
- Button hover effect (background change + subtle scale)

## 📁 Project Structure

- `index.html` — main UI
- `script.js` — app logic
- `src/input.css` — Tailwind input file
- `dist/output.css` — compiled Tailwind output
